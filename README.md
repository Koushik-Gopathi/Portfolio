🚀 My Personal Portfolio
Welcome to my professional portfolio! This website is designed to show who I am, what I can do, and the projects I’ve built. 
It uses a modern "scrolly-telling" style to make browsing my work fun and interactive.


✨ Key Features
1. Interactive Scroll Animation
As you scroll down the home page, you'll see a smooth animation that follows your movement.
This isn't a video—it's a high-performance image sequence that makes the site feel alive and responsive.

3. My Projects (Works)
A clean grid layout where I showcase my best coding projects. Each project has:
A brief description of what it does.The tools I used to build it.
Buttons to View the Code on GitHub or see a Live Demo of the site.

4. Tech Stack (My Tools)
A dedicated section that lists all the programming languages, frameworks, and tools I use for development, organized into categories like Frontend and Backend.

5. My Journey (Timeline)
A professional timeline that tracks my education and career history, from where I started to where I am today.

6. Floating Navigation (The Dock)
A smooth, floating menu at the bottom of the screen that lets you quickly jump to any section of the site (Home, Projects, Skills, etc.).

7. Contact Me
A built-in contact form and social media links (LinkedIn, GitHub, Twitter) so people can easily reach out to me for work or collaboration.


🛠️ Built With
React (JavaScript): The main framework used to build the site.
Next.js: For fast loading and modern web features.
Framer Motion: For all the smooth animations and transitions.
Tailwind CSS: For a clean and modern design.


🚀 How to Run Locally
If you want to run this project on your own computer:
Install dependencies: Run npm install in your terminal.
Start the site: Run npm run dev.
View it: Open http://localhost:3000 in your browser.
